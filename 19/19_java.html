JavaScript Практикум: 20 задач

1. Вывод сообщения

Напишите программу, которая выводит "Hello, World!" в консоль.

2. Сложение двух чисел

Создайте функцию, которая принимает два числа и возвращает их сумму.

3. Проверка четности

Напишите функцию, которая принимает число и возвращает true, если оно четное, и false, если нечетное.

4. Поиск максимального числа

Создайте функцию, которая принимает массив чисел и возвращает максимальное число из этого массива.

5. Сумма элементов массива

Напишите функцию, которая принимает массив чисел и возвращает их сумму.

6. Обратный порядок строки

Создайте функцию, которая принимает строку и возвращает ее в обратном порядке.

7. Проверка палиндрома

Напишите функцию, которая проверяет, является ли строка палиндромом (читается одинаково с обеих сторон).

8. Факториал числа

Создайте функцию, которая вычисляет факториал заданного числа.

9. Генерация случайного числа

Напишите программу, которая генерирует случайное число от 1 до 100 и выводит его в консоль.

10. Сортировка массива

Создайте функцию, которая принимает массив чисел и сортирует его по возрастанию.

11. Удаление дубликатов из массива

Напишите функцию, которая принимает массив и возвращает новый массив без дубликатов.

12. Конвертация градусов Цельсия в Фаренгейт

Создайте функцию, которая принимает температуру в градусах Цельсия и возвращает ее в градусах Фаренгейта.

13. Подсчет гласных букв в строке

Напишите функцию, которая считает количество гласных букв в заданной строке.

14. Создание объекта пользователя

Создайте объект user с полями name, age и email. Выведите его свойства в консоль.

15. Калькулятор

Напишите простую программу-калькулятор, которая выполняет операции сложения, вычитания, умножения и деления двух чисел.

16. Фильтрация массива объектов

Создайте массив объектов с информацией о пользователях (имя и возраст). Напишите функцию для фильтрации пользователей по возрасту (например, старше 18 лет).

17. Проверка наличия элемента в массиве

Напишите функцию, которая проверяет наличие заданного элемента в массиве и возвращает true или false.

18. Слияние двух массивов

Создайте функцию, которая объединяет два массива в один без дубликатов.

19. Калькуляция среднего значения массива

Напишите функцию, которая вычисляет среднее значение элементов массива чисел.

20. Таймер обратного отсчета

Реализуйте простой таймер обратного отсчета на JavaScript с использованием функции setInterval.
